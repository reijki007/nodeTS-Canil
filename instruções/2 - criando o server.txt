1. Criando o servidor
    1.1 - importar as bibliotecas 
        import express from "express";
        import dotenv from 'dotenv';
        import mustache from 'mustache-express';
        import path from 'path';
    
2. Configurando o dotenv
    2.1 - criar arquivo .env
     - configura a porta(80) em .env --pode usar a porta que quiser
    2.2 - dotenv.config(); -- usa o dotenv em server.ts 

3. configurando template engine
    3.1 - criar const server = express();
    3.2 - server.set('view engine', 'mustache'); -- configura o view engine
    3.3 - criar pasta views dentro da pasta do arquivo
    3.3 - server.set('views', path.join(__dirname, 'views')); 
    --pasta de views / (__dirname, 'views')o join pega a pasta do arquivo e junta com os views
    3.4 - server.engine('mustache', mustache()); --roda a função mustache

4. configurando pastas       
    4.1 - criar pasta public
    4.2 - define o caminho static da pasta public 
     - server.use(express.static(path.join(__dirname, '../public')));
    4.3 - roda o server no fim do arquivo server.ts
     - server.listen(process.env.PORT)
    4.4 - criar pasta auxilar (raiz do projeto) _html  
     - mover o arquivo index.html para pasta _html
     - mover as pastas ( css,images) para pasta public
