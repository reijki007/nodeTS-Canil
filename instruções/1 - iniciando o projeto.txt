1. Iniciando o projeto
    1.1 - npm init -> para iniciar o projeto    
    1.2 - tsc --init -> para iniciar o typescript
    1.3 - configurando o typescript(básico){
        habilite "rootDir" modifique para -> "./dist"
        habilite "outDir": modifique para -> "./src"
        habilite "moduleResolution" modifique para -> "node"        
    }

2. instalando Dependencias
    2.1 - npm install express
    2.2 - npm install mustache-express
    2.3 - npm install dotenv -> para as variaveis de ambiente
    ou  - npm install express mustache-express dotenv
    
    ** Dependencias Importantes**
    2.4 - npm install -g typescript 
    2.5 - npm install -g ts-node 
    2.6 - npm install -g nodemon -> monitora os arquivos ( ts,Json, )
    ou  - npm install -g nodemon typescript ts-node

3. instalando Dependencias de desenvolvimento    
    3.1 - npm install --save-dev @types/node
    3.2 - npm install --save-dev @types/express
    3.3 - npm install --save-dev @types/mustache-express

    ou  - npm install --save-dev @types/node @types/express @types/mustache-express

** criar pasta e arquivo - src / server.ts **

4.  criando atalhos para rodar projeto automatico
    1.1 - no Package-json --> criar um atalho em 
    "scripts":{
        "start-dev": "npx nodemon -e ts,json,mustache src/server.ts"
    }
